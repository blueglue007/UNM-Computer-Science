Are YOU ready to party?!

Now that you’ve proven yourself Mr. Krabs would like your help planning all future parties. He always buys appetizers for his parties, but there are often leftovers, and he cannot stand wasting money. SpongeBob and Patrick would probably eat the leftovers like they always do, but Mr. Krabs would prefer to save money.

Program Description

This homework will require that you write a program to calculate the amount of snacks needed for any social events Mr. Krabs hosts. This is useful because planning an event can require a lot of logistics, one of which is knowing how many appetizers to buy.

Capturing user-input

The first thing you’ll need in order to perform your calculations is understanding how to capture user input. Essentially, we want our program to present a nice message to us, and then accept a value that we type and store it in a variable.

Here’s how to store user input in a variable:

some_value = input("Please enter a numeric value: ")

We set a variable equal to what’s called the “input” function. Notice that the string we put inside the parentheses is what prompts us.

Something to note about the variable some_value is that its type is str. We must convert it to a numeric value before we can use it in any arithmetic operations.

To convert a string variable to an int variable, we can do the following:

some_int_value = int(some_value)

Similarly, to convert a string variable to a float variable, we can do the following:

some_float_value = float(some_value)

Your Task

Create directory

  1. Create a hw3 directory inside your existing hw directory.
  2. Navigate into the hw3 directory.
  3. Create a hw3.py file using nano.

Your program

Write code in your hw3.py file that performs the following:

  1. Greet the user.
  2. User input: Store the number of guests in a variable (type should be int).
  3. User input: Store how long the party will last (in hours) in a variable (type should be float).
  4. User input: Store the bite rate (in bites per hour) in a variable (type should be float).
  5. Calculate and print the total appetizer weight in grams.
  6. Calculate and print the number of packages to buy.
  7. Terminate the program with a farewell message.
